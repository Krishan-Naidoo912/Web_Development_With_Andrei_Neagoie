Exercise 1: Includes Method


Copy and paste the below script into your Browser developer console to run.

//#1) Check if this array includes the name "John".
const dragons1 = ['Tim', 'Johnathan', 'Sandy', 'Sarah'];
dragons1.includes("John");

//#2) Check if this array includes any name that has "John" inside of it. If it does, return that
//name or names in an array.
const dragons2 = ['Tim', 'Johnathan', 'Sandy', 'Sarah'];
const newArray = dragons2.filter(name => name.includes("John"))
console.log(newArray);

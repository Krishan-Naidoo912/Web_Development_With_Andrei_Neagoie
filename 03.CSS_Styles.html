Some notes on Elements.

  1. element1 = style element1 only.
  2. element1, element2 = style both element1 and element2 the same.
  3. element1 element2 = style element2 only if it resides inside element1.
  4. element1 > element2 = style element2 only if element1 is the parent of element2.
  5. element1 + element2 = style element2 only if element2 falls after element1.
